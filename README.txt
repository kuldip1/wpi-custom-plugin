===== WPI Custom Plugin =====
Author: http://ignizeo.com/
Donate link: http://ignizeo.com/
Requires at least: 3.4
Tested up to: 4.2.1
Tags: google , custom registration , Custom dashboard

This plugin is used to create a form and create dashboard for Admin user with functionality of google login

== Installation Steps == 

1) Download the WPI Custom Plugin.
2) Unzip the file and upload the wpi-custom-plugin folder to the /wp-content/plugins/ directory using your FTP application.
3) Activate the WPI Custom Plugin through the 'Plugins' menu in WordPress.
4) Once you install plugin it will insert two shortcode with pages as below.
	Create Login page with shortcode [wpi_custom_login]
	Create a Price Form using this shortcode [wpi_custom_form]
5) Once you are done with below step create menu in admin side and assign this pages to menu so it will show in Front side.
